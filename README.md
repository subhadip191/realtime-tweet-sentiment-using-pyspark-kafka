A scalable **real-time tweet sentiment analysis pipeline** built using **Apache Kafka**, **PySpark Structured Streaming**, **Spark NLP**, and **Spark MLlib**. The system processes streaming tweets and classifies them into **four sentiment categories** using semantic embeddings generated by the **Universal Sentence Encoder**.

---

# 📌 Features

- ⚡ Real-time tweet streaming using Apache Kafka
- 🧠 Semantic text embeddings with Spark NLP
- 🤖 Multiclass sentiment classification
- 📊 Live evaluation metrics
- 🔥 Distributed processing with PySpark
- 📈 Continuous streaming inference

# ⚙️ Pipeline

## 1️⃣ Training

- Load first **5,000** tweets
- Generate semantic embeddings
- Train Logistic Regression model
- Save trained pipeline

---

## 2️⃣ Streaming

Producer continuously streams tweets into Kafka.

Consumer:

- Reads Kafka stream
- Predicts sentiment
- Computes

  - Accuracy
  - Precision
  - Recall
  - Cumulative Accuracy

- Saves metrics into

```text
experiment_results.csv
```

## Requirements

- Python 3.10+

```bash
pip install pyspark spark-nlp kafka-python pandas
```

---

# ▶️ Run

### Start Kafka & Zookeeper

# Start Zookeeper
```bash
bin/zookeeper-server-start.sh config/zookeeper.properties
```

# Start Kafka (in a new terminal)
```bash
bin/kafka-server-start.sh config/server.properties
```

# Create the topic (first time only)
```bash
bin/kafka-topics.sh --create --topic tweets --bootstrap-server localhost:9092
```

---

### Start Consumer

```bash
python src/consumer.py
```

---

### Start Producer

```bash
python src/producer.py
```
